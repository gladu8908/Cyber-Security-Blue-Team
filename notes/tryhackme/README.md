# TryHackMe Notes

This folder contains one Markdown file per TryHackMe room.

## Naming convention

Files are named after the room's URL slug, e.g.:

| Room | File |
|---|---|
| SOC Fundamentals | `soc-fundamentals.md` |
| Windows Event Logs | `windows-event-logs.md` |
| Wireshark Basics | `wireshark-basics.md` |

## Note structure

Each file follows the same headings:

1. **Objectives** – what the room says you will learn
2. **Key Concepts** – definitions and explanations in my own words
3. **SOC Workflow** – how the concept fits into a real SOC workflow
4. **Questions / Things to research further** – gaps to close later

> No flags or challenge answers are stored here in line with TryHackMe's terms of service.
